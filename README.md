# 大量客制化寄信系統
**簡介**

影片簡介: https://youtu.be/3Ad-b1-96IU

使用 google 表單後，是否常常不知道如何將特定資料寄給特定使用者?
此寄信系統接受 csv 或 excel 檔
在系統中打上信中的文字，並選擇要替換的欄位
預覽你每一封的信件
確認信件寄出結果

直觀的寄信系統，來使用吧!
